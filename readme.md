## Getting Started

### Install from Logseq Marketplace (ComingðŸ‘·ðŸš§)

- Press [`---`] on the top right toolbar to open [`Plugins`]
- Select `Marketplace`
- Type `File` in the search field, select it from the search results and install

### Usage

#### Trigger `ðŸ’¾ Upload multiple assets`

- From the Block context menu (to open the options on right-click the bullet)
- As slash command

#### Plugin Settings

- Overwrite existing files with the same name: Select
  - `skip`
  - `overwrite`
  - `timestamp`
- Add timestamp to file name: Toggle
  - `true`
  - `false` default
- Enable `ðŸ“‚ Insert multiple files from local folder` block context menu item
  - `true`
  - `false` default

### Option

#### Trigger `ðŸ“‚ Insert multiple files from local folder`

- Embed or link contents without saving them in the assets folder.
