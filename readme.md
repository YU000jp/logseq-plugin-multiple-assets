## Getting Started

âš ï¸Files are saved not in the `assets/` folder but in the `assets/storages/logseq-plugin-multiple-assets/` folder.

### Install from Logseq Marketplace

- Press [`---`] on the top right toolbar to open [`Plugins`]. Select `Marketplace`. Type `Assets` in the search field, select it from the search results and install.

   ![image](https://github.com/YU000jp/logseq-plugin-multiple-assets/assets/111847207/5a3933c0-13f5-4c21-8fc8-c70429d7ad29)

### Usage

#### Trigger `ðŸ’¾ Upload multiple assets`

- From the Block context menu (to open the options, right-click the bullet that has the desired property)
- As slash command

#### Plugin Settings

- Overwrite existing files with the same name: Select
  - `skip` default
  - `overwrite`
  - `timestamp`
- Add timestamp to file name: Toggle
  - `true`
  - `false` default
- Enable `ðŸ“‚ Insert multiple files from local folder` block context menu item
  - `true` default
  - `false`

### Option

#### Trigger `ðŸ“‚ Insert multiple files from local folder`

- Embed or link contents without saving them in the assets folder.
