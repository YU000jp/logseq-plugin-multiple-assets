## Getting Started

### Install from Logseq Marketplace (ComingðŸ‘·ðŸš§)

- Press [`---`] on the top right toolbar to open [`Plugins`]
- Select marketplace
- Type `File` in the search field, select it from the search results and install

### Usage

#### Trigger `ðŸ’¾Insert multiple files into assets`

- From the Block context menu (to open the options on right-click)
- As slash command

#### Plugin Settings

- Overwrite existing files with the same name: Select
  - `skip`
  - `overwrite`
  - `timestamp`
- Add timestamp to file name: Toggle
- `true`
- `false` default
- Enable `ðŸ’¾Insert multiple files from local` block context menu item
- `true`
- `false` default
